--
Este archivo es solo una guía, no lo modifiques.
--

1_ Descarga o convierte tu libro (https://cloudconvert.com/ Recomendado)
2_ Convierte el archivo a JSON (ej.: https://pdfjar.com/pdf-to-json) (Con formato estructurado)
3_ Importa tu archivo JSON a la carpeta "42\media\books"

--
4_ Copia o modifica el JSON para que coincida con el siguiente formato (normalmente solo falta el autor)

{
  "filename": "Título del libro", -- Recomendado --
  "author": "Autor del libro", -- Recomendado --
  "pageCount": Números, -- No obligatorio --
  "pages": [
    {
      "pageNumber": 1, -- No obligatorio --
      "headings": [
        {
          "text": "Primer encabezado", -- Fuente grande, se usa para capítulos. No obligatorio --
          "level": 2
        }
      ],
      "paragraphs": [
        "Texto del libro"
      ]
   },  --> No debe haber espacios y siempre debe haber una coma entre párrafos si quieres añadir o modificar algo.
    {
      "pageNumber": 2,
      "headings": [],
      "paragraphs": [ --> Siguiente sección de párrafos, con o sin encabezados.
        "Texto"
      ]
    },
}  --> Debe terminar de esta manera.

EXTRAS/CONSEJOS:
Las extensiones Enemy Lua y ZedScripts para VS (recomendadas en la wiki oficial) te ayudarán a identificar posibles errores de escritura y sintaxis.
Si quieres hacer un punto y aparte, utiliza \n (ej.: Palabra\nPalabra).
Sin embargo, intenta no usarlo en el título ni en el autor, ya que puede provocar errores visuales.
No te preocupes por la longitud del libro. Al ser solo texto, consume muy pocos recursos y puede manejar libros de hasta 2.000 páginas sin que se note ningún impacto en el rendimiento.
Si quereis ver directamente ejemplos funcionales es recomendable directamente descargar/instalar uno de la PZ Workshop
