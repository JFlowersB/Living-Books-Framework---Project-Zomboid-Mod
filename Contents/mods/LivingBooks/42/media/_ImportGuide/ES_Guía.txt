Importante:
  - Todo lo que se tiene que copiar/modificar de la carpeta _ImportGuide/files del LivingBooksFramework hay que copiarlo en tu nuevo AddonBookMod.
  - Crea el mod con la estructura creada de antemano para la b42 (https://pzwiki.net/wiki/Mod_structure) o simplemente copia tu ModTemplate en tu carpeta Zomboid\Workshop
  - Los archivos de esta guía estan en la carpeta _ImportGuide/files, simplemente cópialos y modificalos se explica más adelante.

Pasos previos:
- Copia el archivo LivingBooksItems.txt y pégalo dentro de tu AddonBookMod aquí --> 42/media/LivingBooksItems.txt
- Modifica y/o copia los bloques de items dentro del archivo para añadir más. Solo tienes que cambiar el nombre del item (ni las propiedades ni el Modulo) Vigila erores de sintaxis.
- Descarga o convierte tu libro (https://cloudconvert.com/) [Gratis sin cuenta]
- Convierte el archivo a JSON (ej.: https://pdfjar.com/pdf-to-json) (Con formato estructurado) [Gratis sin cuenta]
- Importa tu archivo JSON a la carpeta "42\media\books"
- Crea un archivo llamado "books.txt" en la carpeta books --> "42\media\books\books.txt"
- Dentro de books.txt, en una nueva linea añade el nombre del archivo json completo de tu libro importado (ej: libro.json)
- Rellena el "itemFullType" con el nombre del item en "media/LivingBooksItems.txt" (Modulo.Item) que hayas puesto
- Modifica o Arregla el JSON para asegurarte de que coincida con el siguiente formato entre {} (añade el autor si este no esta)

CUIDADO:
- He puesto varios comentarios (-- texto --) para guiarte en el formato. Una vez termines bórralos.

{
  "filename": "Título del libro",                        -- REQUERIDO --
  "itemFullType": "Modulo.NombredelItem",                --REQUERIDO!! Tiene que ser el mismo que el de "42\media\scripts\LivingBooksItems.txt" (Este txt. tiene que estar en la carpeta ADDON)
  "author": "Autor del libro", -- No obligatorio --
  "pageCount": Números, -- No obligatorio --
  "pages": [
    {
      "pageNumber": 1,                                   -- No obligatorio --
      "headings": [
        {
          "text": "Primer encabezado",                   -- Fuente grande, se usa para capítulos. No obligatorio --
          "level": 2
        }
      ],
      "paragraphs": [
        "Texto del libro"
      ]
   },                                                    --> No debe haber espacios y siempre debe haber una coma entre párrafos si quieres añadir o modificar algo.
    {
      "pageNumber": 2,
      "headings": [],
      "paragraphs": [                                    --> Siguiente sección de párrafos, con o sin encabezados.
        "Texto"
      ]
    },
}                                                        --> Debe terminar de esta manera.


Para acabar/Tips:
- Si quieres hacer un punto y aparte, utiliza \n (ej.: Palabra\nPalabra). Sin embargo, intenta no usarlo en el título ni en el autor, ya que puede provocar errores visuales.
- Si añades más de un párrafo o más de una página, recuerda separar cada elemento con una coma y NUNCA dejes espacios vacíos
- Las extensiones Enemy Lua y ZedScripts para VS (recomendadas en la wiki oficial) te ayudarán a identificar posibles errores de escritura y sintaxis.
- No te preocupes por la longitud del libro. Al ser solo texto, consume muy pocos recursos y puede manejar libros de hasta 2.000 páginas sin que se note ningún impacto en el rendimiento.

Recuerda:
- Cada libro es un archivo book.json. Si quieres añadir varios libros, crea varios mods pequeños de contenido o haz un pack de libros en uno solo.
- itemFullType es el ÚNICO elemento que vincula este JSON con un objeto del juego. LivingBooks busca, al iniciar, todos los archivos media/books/book.json de los mods activados y los asocia con el objeto del inventario cuyo FullType coincida exactamente.
- Si quereis ver directamente ejemplos descarga mis libros compatibles en la workshop.
